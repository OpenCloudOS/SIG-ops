# OpenCloudOS运维工具SIG
## 职责范围 Scope
OpenCloudOS作为一个操作系统项目，离不开运维工具的建设。本小组的目标致力于，将腾讯云的运维经验中积累的有较广泛价值工具，以开源的形式贡献给社区。主要工作包括对现有系统开源工具及命令的能力补充和新工具能力的开发和运营。
